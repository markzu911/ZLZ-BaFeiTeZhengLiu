# 巴菲特人物时间线调研

## 一、早年经历 (1930-1950)

| 时间 | 事件 | 对思维的影响 |
|------|------|------------|
| 1930年8月30日 | 出生于内布拉斯加州奥马哈 | — |
| 1930s | 父亲Howard Buffett是共和党国会议员、商人 | 诚信、勤奋、财务纪律的启蒙；"建立声誉需要20年，毁掉只需5分钟"来自父亲 |
| ~1940年 | 10岁时读到父亲办公室的投资书籍 | 对数字和复利的早期痴迷 |
| 1942年 | 11岁买入人生第一只股票（Cities Service Preferred，3股） | 早期经历教会了耐心——他在小赚后卖出，错过了后来的大涨 |
| ~1943年 | 13岁开始送报纸（Washington Post） | 清晨4:30起床送报，同时观察到可以卖杂志订阅赚外快；到15岁时赚了2000美元 |
| 1945年 | 15岁，与朋友合伙25美元买弹珠机放理发店 | 第一次真正的商业经营：虚构"Wilson先生的投币机公司"推销，展示了早期的商业直觉和杠杆思维 |
| 1946年 | 用送报赚的钱买了40英亩农地 | 16岁时已有约53,000美元积蓄（相当于今天60万+美元） |
| 1947年 | 以1200美元卖掉弹珠机生意 | 后来跟盖茨说"这是我做过的最好的生意，我的商业生涯在很早就达到了巅峰" |

## 二、教育与师承 (1947-1956)

| 时间 | 事件 | 对思维的影响 |
|------|------|------------|
| 1947-1949 | 宾夕法尼亚大学沃顿商学院（转学） | 觉得课堂教育不如实际经验有价值 |
| 1950 | 内布拉斯加大学林肯分校毕业 | 19岁完成本科 |
| 1950 | 读到Benjamin Graham的《聪明的投资者》 | "这是我读过的最好的投资书"——奠定了价值投资的基础框架 |
| 1950 | 报名Dale Carnegie沟通课程 | 彻底改变了社交和表达能力；办公室里挂的是Carnegie毕业证而非大学文凭 |
| 1951 | 哥伦比亚大学商学院硕士，师从Benjamin Graham | 直接接触Graham的思维方式，是唯一在Graham课上拿A+的学生 |
| 1951-1954 | 在父亲的公司Buffett-Falk & Co做投资销售员 | — |
| 1954-1956 | 在Graham-Newman Corp做证券分析师 | 年薪12,000美元；在Graham手下学习评估企业清算价值；与Walter Schloss共事 |
| 1956 | Graham退休，Graham-Newman解散 | 巴菲特选择回到奥马哈而非留在纽约——远离华尔街的物理距离成为独立思考的结构性优势 |

## 三、合伙基金时期 (1956-1969)

| 时间 | 事件 | 对思维的影响 |
|------|------|------------|
| 1956年5月 | 25岁创建Buffett Partnership Ltd.，初始资金105,100美元 | 合伙协议体现核心原则：利益一致、长期导向 |
| 1957年 | 管理5个合伙基金 | — |
| 1959年 | 首次见到Charlie Munger | 终身伙伴关系的起点；"我认识Charlie后，不再是同一个投资者了" |
| 1962年 | 开始买入Berkshire Hathaway（纺织公司）的股票 | — |
| 1964年 | 父亲Howard Buffett去世 | — |
| 1965年 | 取得Berkshire Hathaway控制权 | 后来承认这是情绪驱动的错误——因为对管理层生气而收购了一家衰落的纺织厂 |
| 1967年 | 以860万美元收购National Indemnity保险公司 | **关键转折点**：开启保险浮存金模式，奠定Berkshire帝国的资金基础 |
| 1969年 | 在市场狂热中关闭合伙基金 | 极其罕见的自律：在最容易赚钱的时候选择退出——"我找不到符合标准的投资了" |

## 四、Berkshire Hathaway帝国建设 (1970-2000)

| 时间 | 事件 | 对思维的影响 |
|------|------|------------|
| 1972年 | 以2500万美元收购See's Candies | **思想转折点**：第一次为品质付溢价，从Graham烟蒂股转向芒格品质投资 |
| 1973年 | 开始买入Washington Post股份 | 与Katharine Graham建立终身友谊；低估值+优秀管理层 |
| 1976年 | 在GEICO接近破产时大举买入 | 45年后全资收购——展示了超长时间跨度的投资能力 |
| 1977年 | 与妻子Susan分居（未离婚） | 个人生活转折；Susan介绍Astrid Menks给巴菲特 |
| 1985年 | 关闭Berkshire纺织业务 | 终于承认无法在衰落行业中挣扎 |
| 1988年 | 以10.23亿美元买入可口可乐6.2%股份 | **标志性投资**：全球品牌护城河的最佳案例 |
| 1993年 | 收购Dexter Shoe（用BRK股票支付） | **最大失误**：后来称之为"worst deal I've ever made"，机会成本120亿+ |
| 1996年 | 全资收购GEICO | 完成45年的投资闭环 |
| 1998年 | 收购General Re | 发现衍生品隐藏风险，加深对衍生品的警惕 |
| 1999年 | 互联网泡沫期间拒绝投资科技股 | 被媒体嘲笑"落伍"，但坚守能力圈原则 |

## 五、新千年与演化 (2000-2019)

| 时间 | 事件 | 对思维的影响 |
|------|------|------------|
| 2001年 | 互联网泡沫破裂，巴菲特被证明正确 | "只有退潮时，才知道谁在裸泳" |
| 2004年 | 妻子Susan去世 | 深刻的个人损失 |
| 2006年 | 宣布将大部分财富捐给盖茨基金会 | **价值观转折**：从计划死后捐赠到在世时大量捐赠 |
| 2006年 | 与Astrid Menks结婚 | — |
| 2008年 | 金融危机中写NYT文章"Buy American. I Am." | 在极端恐慌中展示逆向投资信念 |
| 2008年 | 通过芒格推荐投资BYD | 愿意在芒格强烈推荐时超出自己的能力圈 |
| 2010年 | 与盖茨共同发起Giving Pledge | 推动亿万富翁承诺捐出至少一半财富 |
| 2011-2018年 | 投资IBM后全部清仓 | 违反能力圈原则的教训 |
| 2015年 | 与3G Capital合作Kraft Heinz收购 | 后来成为重大失误——市值蒸发约570亿 |
| 2016年 | 大举买入Apple | **晚年最重大决策**：将Apple视为消费品而非科技公司 |

## 六、晚年与传承 (2020-2026)

| 时间 | 事件 | 对思维的影响 |
|------|------|------------|
| 2020年 | COVID期间清仓所有航空股 | 承认航空业投资是重复犯错 |
| 2020年 | 开始秘密买入日本五大商社 | 90岁仍在学习新市场；低估值+日元套利结构 |
| 2023年11月 | Charlie Munger去世（享年99岁） | 失去60年合作伙伴，Berkshire进入"后芒格时代" |
| 2024年 | 大幅减持Apple（从50%+降至约25%） | 市场解读分歧：税务规划vs估值担忧 |
| 2024年 | 秘密建仓Chubb保险（约67亿美元） | 继续深化保险布局 |
| 2024年Q3 | Berkshire停止回购自家股票 | 信号：连巴菲特自己都觉得BRK股价偏高 |
| 2024-2025年 | 现金储备创纪录（超3340亿美元，后达3817亿美元） | 对市场整体估值的隐性看空 |
| 2025年3月 | 增持日本五大商社至约10%（每家8.5-9.8%） | 日本投资组合总值超300亿美元，5年回报392% |
| 2025年3月 | 最后一次以CEO身份接受CNBC采访 | 讨论退休安排和Berkshire未来 |
| 2025年5月 | 最后一次以CEO身份主持股东大会 | 宣布2025年底退休 |
| 2025年底 | 正式退休，Greg Abel接任CEO | 结束60年掌舵Berkshire |
| 2026年2月 | 日本投资6年累计获利240亿美元 | — |

## 七、思想演化关键转折点总结

| 转折点 | 时间 | 从 | 到 | 驱动力 |
|--------|------|-----|-----|--------|
| 价值投资启蒙 | 1950 | 商业直觉 | Graham框架 | 《聪明的投资者》 |
| 品质投资转型 | 1972 | 烟蒂股 | 优秀企业+合理价格 | See's Candies + 芒格影响 |
| 保险帝国建设 | 1967 | 传统投资 | 浮存金杠杆模式 | National Indemnity收购 |
| 极端自律 | 1969 | 高速增长 | 主动退出 | 找不到好机会 |
| 科技态度软化 | 2016 | 完全回避科技 | 选择性拥抱 | Apple（重新定义为消费品） |
| 慈善转型 | 2006 | 死后捐赠 | 在世大量捐赠 | 盖茨基金会影响 |
| 全球化视野 | 2020 | 美国中心 | 全球机会 | 日本商社投资 |

## 八、个人生活关键线索

### 家庭
- 与Susan Thompson (1952年结婚, 1977年分居, 2004年去世) 的复杂关系
- 与Astrid Menks (1977年开始同居, 2006年结婚) 的稳定关系
- 三个子女：Susan, Howard, Peter
- 传记揭示：工作痴迷导致家庭关系紧张

### 生活方式
- 至今住在1958年以31,500美元买的奥马哈房子
- 饮食习惯极度简单（可乐、汉堡、冰淇淋）
- 桥牌爱好者（与盖茨等人定期打桥牌）
- 公众节俭形象 vs 私人飞机等精英生活方式的矛盾

### 与Bill Gates的关系变化
- 曾是最亲密的朋友和桥牌搭档
- 因Epstein关联事件切断联系
- 揭示巴菲特对声誉风险的零容忍
